# Contributing to whisper-key

## Contributor License Agreement (CLA)

By submitting a contribution, you grant Nicolas Verki (NVK Labs) the perpetual,
worldwide, non-exclusive right to use your contribution under both the GPLv3
and Commercial licenses. This allows the dual-license model to remain sustainable.

Your contribution must be your original work or properly attributed.

## How to Contribute

1. Fork the repo
2. Create a branch: `git checkout -b feature/my-feature`
3. Commit: `git commit -m "feat: description"`
4. Push and open a Pull Request

## Code Style

- Files < 300 lines
- No docstrings, no unnecessary comments
- Explicit variable and function names
- Test before committing
